== Description ==

 This is a WordPress plugin for people who use the most popular NextGEN gallery and do care about website optimization,
 speed and page score. It should serve as a swiss-knife for image optimization for your website.

 Result: Your WordPress blog will load much faster ! Google Page Speed will recognize all your images as 100% optimized, which will give you better score and page rank.

 In this version that means:

  * Much faster loading of images (check Settings tab, turn on 85/100 + progressive check boxes)
  * Better page speed score for your website
  * Leads to overall better SEO site score and ranking

= Supports =

  * NextGEN, the most pouplar WordPress gallery
  * WordPress Media Library (in commerical version)
  * Active theme images optimization (in commercial version)

= Features =

 * Optimize all images in NextGEN Gallery (including thumbnails)
 * Automatic images optimization in NextGEN WordPress Gallery Plugin:

   * on image upload
   * on thumbnail generation
   * on thumbnail edit or rotate action

== Installation ==

1. Download, upload and install .zip under Plugins >> Add New > Upload, and activate the NextGEN Gallery Image Optimizer WordPress plugin.
2. From your Wordpress Dashboard, go to Tools > NextGen Gallery Image Optimizer > Follow the on-screen cues.

That's it ... have fun! For more information, feel free to visit the official website at <a href="http://goo.gl/b15sQ8" target="_blank">www.ngimo.com</a> or watch demo video at http://www.ngimo.com/demo
